战前心理

在此我们使用hexo+GitHub来搭建自己的博客，为什么要使用Hexo? Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。但仅使用Hexo只能通过本地看到我们的网站，为了让我们的网站能在互联网上看到，所以我们需要将博客部署到GitHub上。

战前准备

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

  • Node.js
  • Git

安装Node.js & Git

安装Node.js有很多种方法，比如通过官网安装。这里使用drew进行下载并安装。

安装drew

ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装nodejs

brew install nodejs

安装git

brew install git

安装 Hexo

新建一个专门存放博客的目录，进行一下操作。

npm install hexo-cli -g
hexo init blog
cd blog
npm install

启动服务

hexo server

打开http://localhost:4000就可以看到本地网站。
